# Lessons Learned

## Mongoose

- Learned that models are registered only when their files are executed.
- Understood why `import type` does not register models.
- Learned two approaches to model registration:
  - Import related models.
  - Central `models/index.ts`.

## Next.js

- Learned how Server Actions differ from API Routes.
- Learned why caching can cause stale data.

## TypeScript

- Learned the difference between type imports and runtime imports.
